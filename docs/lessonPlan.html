Minecraft Plugin Creation with Scratch

Description

This course will guide the student step-by-step to the completion of 3 plugin minigame projects.
Each project will highlight different minecraft features such as joining a server, creating a lobby, a scoreboard, determining a game winner, special weapons and many other minecraft properties and capabilities.
Ultimately each student will be shown how to populate and administrate their own public server that will be available to any player with an internet connection.

Project 1 Spleef MiniGame

Highlights

  • Install your own minecraft server on your personal computer
  • Creating a lobby and spawn point
  • Select team color and prevent friendly damage
  • Protect certain types of blocks from destruction
  • Detect game winner
  • Reset the game world
  • Setup a public web server

Syllabus

ClassDescription
1Install server and blocklyScriptcraft on student's personal computer (windows or MAC),Overview of Game
2Special weapon modification: snowballs and create scoreboard for game
3Protect all blocks except snow,Randomize chest contents
4Set spawn point and gear when on respawn and when player joins the server, determine if this is the first time the player has joined the game in progress
5Create Spleef lobby, Select team color at lobby and protect from friendly damage
6Teleport player to team spawn point
7Game over detection, Reset the game
8How to setup up a public web server
