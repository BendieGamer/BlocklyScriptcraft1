Bay Coding Club Lesson Plan

Project 1 Spleef MiniGame

ClassDescription
1Install server and blocklyScriptcraft on student's personal computer (windows or MAC)
2Overview of Game, Exploding snowballs and create scoreboard for game
3Protect all blocks except snow
4Set spawn point and gear when on respawn and when player joins the server
5Create Spleef lobby
6Select team color at lobby and protect from friendly damage
7Game over detection, Resetting the gamet
8Setting up a public web server for spleef (number of player supported, speed required)

Project 2 Omaha Beach MiniGame

ClassDescription
1Special Rules and overview of game
2Lobby creation and team selection/locations
3Cobweb damage, Saving gear on respawn
4Set spawn point and gear when player joins the server
5Player types: King, soldier
6Special weapon: sniper arrows
7Special weapon: minigun
8Game Over detection

Project 3 Tower Defense Minigame

ClassDescription
1Special Rules and overview of game
2Lobby creation and team selection/locations
3Scoreboard creation
4Hopper emerald drop linked to scoreboard
5Teleportation based on sign click
6Protection from friendly team damage
7Use redstone and command blocks to drop emeralds
8Game Over detection
