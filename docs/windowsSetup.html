Windows 10 Setup

Git

  • Download 64-Bit Git for windows: https://git-scm.com/download/win
  • Run the git install executable

Local Minecraft Server

  • Open a cmd window
    • Hold the Windows key and press the r key
    • Enter the command: cmd
  • Enter the command:cd c:\
  • Enter the command:git clone https://www.github.com/Paulware/MinecraftServer

    BlocklyScriptcraft

  • Open a cmd window
    • Hold the Windows key and press the r key
    • Enter the command: cmd
  • Enter the command:cd c:\
  • Enter the command:git clone https://www.github.com/Paulware/blocklyScriptcraft
    • After cloning the git repository, you can run blockly Scripcraft by opening C:\BlocklyScriptcraft\blocklyScriptcraft.html using google chrome (IE will not work)
