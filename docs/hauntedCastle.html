Haunted Castle

Let's make a castle that has a mob living in it.
The command:
/js castle ()
will create a castle with 4 towers. We will create a player-moved listener event which will spawn a mob when the player entires one tower
Use the F3 built in feature to look at the top ladder block for one of the towers.
