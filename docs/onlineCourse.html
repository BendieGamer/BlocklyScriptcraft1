Create Minecraft Plugin using Scratch

This course leverages the students knowledge of minecraft and scratch programming to permit them to create their own minecraft plugin
Most students have already been exposed to scratch programming and the MOJANG minecraft game engine. Combining the two, provides a "familiar" context that the students can use to develop their own personal game.

Pre-Requisites

Before registering students should have:
  • A desktop, laptop, or MAC capable of running minecraft
  • A MoJang minecraft account
  • A microphone and speakers (or headset) connected to their computer
  • 64-bit git for windows installed from: https://git-scm.com/download/win

Notes

Suggested age group: 8 to 16 years old
